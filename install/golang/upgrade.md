## Upgrade Or Downgrade

1. Find version url
2. Download

   ```bash
   wget https://studygolang.com/dl/golang/go1.14.4.linux-amd64.tar.gz # Change version url

   ```

3. Unzip to go path
   ```bash
   sudo tar -xzf go1.14.4.linux-amd64.tar.gz -C /opt # Unzip to /opt/go - this is the go root path.
   ```
