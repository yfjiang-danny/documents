### Path commands
**Uppercase specifies absolute coordinates, lowercase specifies relative coordinates**
1. `M` - move to
2. `L` - line to
3. `H` - draw horizontal line
4. `V` - draw vertical line
5. `Z` - close path
6. `C` - Cubic Beziers
7. `Q` - Quadratic
8. `S` - several Cubic Beziers
9. `A` - arcs
